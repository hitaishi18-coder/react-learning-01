✅ Radio Buttons & Dropdown in React

🧠 About the Project
This app teaches:

📻 How to handle radio buttons in React

📍 How to manage select/dropdown values

🎯 Controlled components using useState

💡 Live UI updates based on user input

🛠 Tech Stack
⚛️ React (Vite)

🔁 React Hooks (useState)

💅 CSS for styling
